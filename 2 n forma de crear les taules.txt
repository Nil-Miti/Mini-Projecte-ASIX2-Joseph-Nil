
Para crear una base de datos en MariaDB, puedes utilizar el cliente de línea de comandos o una herramienta de administración gráfica como MySQL Workbench. A continuación, te proporcionaré instrucciones para crear una base de datos utilizando el cliente de línea de comandos.



Crear la base de datos:
Una vez que hayas ingresado la contraseña, puedes crear una nueva base de datos con el siguiente comando. Reemplaza "nombre_de_base_de_datos" con el nombre que desees para tu base de datos.



CREATE DATABASE nombre_de_base_de_datos;


Seleccionar la base de datos:
Para comenzar a trabajar con la base de datos que acabas de crear, selecciona la base de datos con el siguiente comando:


USE nombre_de_base_de_datos;

Crear tablas y definir la estructura:
Ahora puedes crear tablas y definir la estructura de tu base de datos. 

Aquí tienes un ejemplo básico de cómo puedes crear una tabla:


CREATE TABLE nombre_de_tabla (
    id INT PRIMARY KEY,
    nombre VARCHAR(255),
    edad INT
);
Este ejemplo crea una tabla llamada "nombre_de_tabla" con columnas para el ID, el nombre y la edad.



Salir de MariaDB:
Cuando hayas terminado de trabajar en la base de datos, puedes salir de MariaDB con el siguiente comando:


EXIT;
